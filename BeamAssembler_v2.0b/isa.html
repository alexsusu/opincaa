nop - No operation, PC advance

Control instructions
ajmp [Jump Index] - Absolute jump to specified jump index
call [Jump Index] - Subroutine call to specified jump index (also does a push)
arjmp [Register ID] - Absolute jump with register
ret - Return from subroutine (also does a pop)
halt - Halt (no PC advance)
rjmp [Jump Index] - Relative jump to specified jump index
cjmp [Register ID] [Jump Index] - Conditional jump if condition register value is NOT 0, to the specified jump index
ncjmp [Register ID] [Jump Index] - Conditional jump if condition register value is 0, to the specified jump index

Thread control instructions
remThn [Address] - Kills the thread n, dumps the context into the memory stack and writes the stack tail address to the specified memory address(n = 0..7)
addThn [ContextAddress] - activate the thread n with the specified stack tail address (n = 0..7)

Data instructions
inc [Destination Register ID] [Source Register ID] - Increments the value from the source register and writes it in the destination register
dec [Destination Register ID] [Source Register ID] - Decrements the value from the source register and writes it in the destination register
add [Destination Register ID] [Source Register ID] [Source Register ID] - Adds the value from the two source registers and writes it in the destination register
sub [Destination Register ID] [Source Register ID] [Source Register ID] - Subtracts the value of the 2nd source register from the value of the 1st source register and writes the result in the destination register
move [Destination Register ID] [Source Register ID] - Copies the value from the source register in the destination register
rotate [Destination Register ID] [Source Register ID] - Rotates the value from the source register 1 bit to the left and writes the result in the destination register
chsgn [Destination Register ID] [Source Register ID] - Changes the sign of the value from the source register and writes the result in the destination register
asr [Destination Register ID] [Source Register ID] - Arithmetically rotates the value from the source register 1 bit to the right and writes the result in the destination register
bwnot [Destination Register ID] [Source Register ID] - Bitwise NOT on the value from the source register and writes the result in the destination register
bwand [Destination Register ID] [Source Register ID] [Source Register ID] - Bitwise AND between the values of the 2 source registers and writes the result in the destination register
bwor [Destination Register ID] [Source Register ID] [Source Register ID] - Bitwise OR between the values of the 2 source registers and writes the result in the destination register
bwxor [Destination Register ID] [Source Register ID] [Source Register ID] - Bitwise XOR between the values of the 2 source registers and writes the result in the destination register
lsr [Destination Register ID] [Source Register ID] - Logically shifts 1 bit right the value from the source register and writes it in the destination register
mul [Destination Register ID] [Source Register ID] [Source Register ID] - Multiplies the value from the two source registers and writes it in the destination register
val [Destination Register ID] [Numerical Value] - Loads the specified 16 bit unsigned value and writes it in the destination register
sval [Destination Register ID] [Numerical Value] - Loads the specified 16 bit signed value and writes it in the destination register

Test instructions
zero [Destination Register ID] [Source Register ID] - Tests if the value from the source register is 0 and writes the result (1 or 0) in the destination register
eq [Destination Register ID] [Source Register ID] [Source Register ID] - Tests if the values from the source registers are equal and writes the result (1 or 0) in the destination register
lt [Destination Register ID] [Source Register ID] [Source Register ID] - Tests if the value from the 1st source register is less than the value from the 2nd source register and writes the result (1 or 0) in the destination register
gt [Destination Register ID] [Source Register ID] [Source Register ID] - Tests if the value from the 1st source register is greater than the value from the 2nd source register and writes the result (1 or 0) in the destination register
addCr [Destination Register ID] [Source Register ID] [Source Register ID] - Adds the values from the 2 source registers and writes the carry bit (1 or 0) in the destination register
subCr [Destination Register ID] [Source Register ID] [Source Register ID] - Substracts the values from the 2 source registers and writes the carry bit (1 or 0) in the destination register
incCr [Destination Register ID] [Source Register ID] - Increments the value from the source register and writes the carry bit (1 or 0) in the destination register
decCr [Destination Register ID] [Source Register ID] - Decrements the value from the source register and writes the carry bit (1 or 0) in the destination register

Input-output instructions
push - Pushed the current context onto the memory stack
pop - Pops a context from the top of the memory stack into the current context
load [Address Register ID] [Destination Register ID] - Loads the value from the memory, at address specified by the address register, and writes it in the destination register
store [Address Register ID] [Value Register ID] - Writes the value from the value register in the memory, at the address specified by the address register

Additional info:
  • A jump index is specified in the following way:
    index_name:
    where the index_name is a standard alphanumerical indentifier
  • A register ID has the following form: Rn or rn, with n = 0..15; NOTE that R0 is the PC register
  • A new code segment address is specified in the following way:
    (numerical_value)
    where the numerical_value can be specified in decimal base by simply writing the value, or in hexadecimal base, by using 0x as a prefix.
